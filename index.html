How it works

Seasonal themes

CSS variables power gradients and accents. Themes change automatically by date and can be forced with ?theme= in the URL.

Canvas overlays

Snow and heart particles run on a single canvas for performance. Use the overlay toggle in the header to pause animations.

Gallery growth

Edit gallery.js to add more photos. The masonry-like grid stays responsive, and images lazy-load for efficiency.

Music memory

LocalStorage remembers track choice, volume, and whether you left playback on, while respecting the browser's autoplay rules.
