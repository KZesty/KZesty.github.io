How themes work

The page reads a theme query string and falls back to seasonal defaults: Christmas in December, Valentine’s through February 20, otherwise the calm default palette. Switching themes updates the URL so you can share the view.

Overlays

Snowflakes drift down for Christmas while hearts rise for Valentine’s. Use the overlay controls in the banner or music card to pause the motion.

Shared album

If you set VITE_ICLOUD_EMBED_URL in the inline config, an embedded album fills the shared album card. Without it, a friendly placeholder encourages you to add the link.
